# LAXlerbo.github.io

### 本地预览
```bash
`hexo generate`
`hexo server`
```

### 部署
```
`hexo generate`
`hexo deploy`

//等于一次性执行了，清空、刷新、部署三个命令
`hexo clean && hexo g && hexo d`

```

### Tips
```
// hexo支持更加简单的命令格式了，比如：
hexo g == hexo generate
hexo d == hexo deploy
hexo s == hexo server
hexo n == hexo new
```
